## Install Any Skill Manually

Each skill follows the [Agent Skills](https://github.com/anthropics/skills) open standard. Copy the folder into your project's `skills/` directory:

```bash
git clone https://github.com/koinod/skills.git
cp -r skills/sales-call-intelligence ./skills/
```

Or install via ClawHub:
```bash
clawhub install koino-sales-call-intelligence
```
