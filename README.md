# KOINO Skills

**55+ production-grade AI agent skills for Claude Code, OpenClaw, Codex, and ChatGPT.** Each skill is a drop-in module that gives your AI agent a real capability — sales call analysis, deep research, content repurposing, fleet coordination, and more. Free lite editions included. Full alpha versions at [koino.capital/kits](https://koino.capital/kits).

Built by [KOINO Capital](https://koino.capital) — the operator-first AI agency.

---

## Quick Install

```bash
# Via ClawHub (recommended)
clawhub install koino-sales-call-intelligence

# Or clone and copy
git clone https://github.com/koinod/skills.git
cp -r skills/sales-call-intelligence ./skills/
```

Each skill follows the [Agent Skills](https://github.com/anthropics/skills) open standard.

## Contributing

We accept skill contributions. To submit:

1. Fork this repo
2. Add your skill in its own directory with a `README.md` and `skill.md`
3. Follow the [Agent Skills](https://github.com/anthropics/skills) open standard
4. Open a PR with a one-line description of what the skill does
