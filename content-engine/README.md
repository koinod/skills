# Content Engine Lite

Free content repurposing skill for Claude Code / OpenClaw.

**One source in. Five clips out. Two platforms. Zero cost.**

## What It Does

Drop in a transcript, blog post, or any raw content. Content Engine Lite extracts the 5 best moments, applies proven hook formulas, and formats them for Instagram Reels and LinkedIn -- with a 3-post/week calendar.

## Quick Start

```bash
# Basic usage
./scripts/repurpose.sh --source transcript.txt --clips 5

# With brand voice
./scripts/repurpose.sh --source transcript.txt --brand-name "Your Brand" --tone "direct, edgy" --topics "sales, growth"
```

## What's Included

- 5 clips per source file
- 10 hook formulas (bold claim, pattern interrupt, confession, etc.)
- Instagram Reels formatting (script, caption, cover concept)
- LinkedIn post formatting (micro-lesson structure)
- 3-post/week content calendar

## Requirements

- Ollama with `qwen2.5:3b` (or any local model)
- jq
