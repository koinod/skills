## Instagram Reels

### Technical Specs
| Spec | Value |
|------|-------|
| Aspect Ratio | 9:16 (vertical) |
| Resolution | 1080 x 1920 px |
| Length | 15-90 seconds (sweet spot: 30-60s) |
| Caption Limit | 2,200 characters |
| Hashtag Limit | 30 (recommend 20-25) |
| Cover Image | 1080 x 1920 px |
| File Size | Max 4GB |
| Format | MP4, MOV |

### Algorithm Signals (2026)
1. **Shares via DM** — Now the #1 ranking signal. Content that gets sent to friends gets 3-5x more reach.
2. **Watch time / completion rate** — Videos watched to the end get pushed harder.
3. **Saves** — Strong signal of value. Instagram shows saved content to similar users.
4. **Audio clarity** — Instagram's AI listens to speech for topic categorization. Clear voiceovers with keyword-rich language improve search ranking.
5. **Authenticity score** — Overly polished/ad-like content gets suppressed. Raw, human content wins.
6. **Engagement velocity** — First 30 minutes matter most. Post when your audience is online.

### Best Practices
- Open with motion in the first frame (no static title cards)
- Use captions — 85% watch without sound
- Front-load the hook in both video AND caption
- CTA in the last 3 seconds AND caption
- Reply to every comment in first hour
- Use carousel format for step-by-step content (up to 10 slides, 1080x1350 or 1080x1080)
- Stories: 1080x1920, 15 sec max per frame, use polls/questions for engagement

### Optimal Posting Times
- Weekdays: 7-8am, 12-1pm, 7-9pm (user's local timezone)
- Best days: Tuesday, Wednesday, Thursday
- Avoid: Saturday morning, Monday before 10am

## TikTok

### Technical Specs
| Spec | Value |
|------|-------|
| Aspect Ratio | 9:16 (vertical) |
| Resolution | 1080 x 1920 px |
| Length | 15 seconds - 10 minutes (sweet spot: 30-60s) |
| Caption Limit | 4,000 characters |
| Hashtag Limit | No hard limit (recommend 3-5) |
| File Size | Max 287MB (mobile), 512MB (desktop) |
| Format | MP4, MOV |

### Algorithm Signals (2026)
1. **Watch time + replay rate** — The dominant signal. Videos rewatched get exponential reach.
2. **Predictive AI** — TikTok now surfaces content users haven't searched for but will likely engage with. Niche down hard.
3. **Comment-to-view ratio** — High comment ratio = controversial/engaging = more push.
4. **Shares** — Especially shares to external (SMS, other apps).
5. **Shopping integration** — Content using TikTok Shop features gets algorithmic priority.
6. **Keyword-rich captions** — TikTok's search is becoming a Google competitor. Treat captions like SEO.

### Best Practices
- 3-5 hashtags MAX — hashtag stuffing actively hurts reach in 2026
- Use keywords in captions as if writing for search
- Pattern interrupt in first 1 second (not 3 — TikTok moves faster)
- Stitch and Duet formats get 2x organic reach
- Post 1-3x daily for growth phase, 1x daily for maintenance
- Use trending sounds but add original commentary
- End with a loop (last frame connects to first frame for replays)
- Text on screen throughout (not just captions)

### Optimal Posting Times
- Weekdays: 10-11am, 2-3pm, 7-9pm
- Best days: Tuesday, Thursday, Friday
- Weekend: 11am-1pm works well for lifestyle/entertainment

## LinkedIn

### Technical Specs
| Spec | Value |
|------|-------|
| Text Post | 3,000 characters (ideal: 1,000-1,300) |
| Image | 1200 x 627 px (landscape) or 1080 x 1080 (square) |
| Video | 16:9 or 1:1, max 10 min, 5GB |
| Carousel/Document | PDF, max 300 pages, 100MB |
| Hashtags | Max 3 (more = spammy) |
| Article | No limit |

### Algorithm Signals (2026)
1. **Dwell time** — How long people spend reading your post. Long-form that holds attention wins.
2. **Meaningful comments** — One thoughtful comment > 50 likes. Comments with 10+ words signal quality.
3. **Saves/bookmarks** — Massive signal for "professional value" content.
4. **Professional relevance** — LinkedIn's AI categorizes content by professional topic. Stay in your lane.
5. **Native content preference** — Links to external sites get 50-70% less reach. Keep people ON LinkedIn.
6. **Document/carousel engagement** — Slide-through rate on documents is a strong signal.

### Best Practices
- Hook in the first 2 lines (before "see more" truncation)
- Use line breaks liberally — wall of text = scroll past
- "Micro-lesson" format: Hook → Story/Context → Insight → Actionable Takeaway → Question
- End with a question to drive comments (not engagement bait — real questions)
- No hashtags or max 3 professional/industry ones
- Post as text first, add image in comment if needed (some tests show text-only gets more reach)
- Document/carousel posts: 8-15 slides, one big idea per slide, 30pt+ font
- Never start with "I'm excited to announce" — it's the LinkedIn equivalent of clearing your throat

### Optimal Posting Times
- Tuesday-Thursday: 7-8am, 12pm
- Best single slot: Wednesday 12pm
- Avoid: Friday afternoon, weekends (B2B audience is offline)

## X (Twitter)

### Technical Specs
| Spec | Value |
|------|-------|
| Tweet | 280 characters (Premium: 4,000) |
| Thread | Unlimited tweets |
| Image | 1600 x 900 px (16:9) or 1080 x 1080 |
| Video | 16:9 or 1:1, max 2:20 (Premium: 60 min) |
| GIF | Max 15MB |
| Hashtags | 1-2 max |

### Algorithm Signals (2026)
1. **Replies and quote tweets** — Conversation starters get amplified.
2. **Bookmark rate** — Save-worthy content gets pushed to "For You" tab.
3. **Link-out penalty** — External links still get suppressed. Put the value IN the tweet.
4. **Time on post** — How long someone spends reading your tweet/thread.
5. **Profile clicks** — Tweets that drive profile visits signal "interesting person."
6. **Premium subscriber boost** — Verified/Premium accounts get 2-4x more initial impressions.

### Best Practices
- Single tweet: One bold idea. No filler words. Every word earns its place.
- Thread structure: Hook tweet → 5-7 value tweets → CTA tweet
- Thread hook format: "I [did thing] in [timeframe]. Here's the playbook: (thread)"
- Use "Quote tweet this with your take" as CTA
- Engage in replies to larger accounts in your niche (visibility hack)
- Repost your best-performing tweets every 30-60 days
- Screenshot your tweet as an image post on other platforms (cross-pollination)
- Spaces (live audio) → clip the recording → repurpose

### Optimal Posting Times
- Weekdays: 8-10am, 12-1pm, 5-6pm
- Best days: Monday, Tuesday, Wednesday
- Real-time events: Post during industry events/conferences for trending boost

## YouTube Shorts

### Technical Specs
| Spec | Value |
|------|-------|
| Aspect Ratio | 9:16 (vertical) |
| Resolution | 1080 x 1920 px |
| Length | Up to 60 seconds (sweet spot: 30-58s) |
| Title | Max 100 characters |
| Description | 5,000 characters |
| Hashtags | 3-5 in description |
| File Size | No official limit |
| Format | MP4, MOV |

### Algorithm Signals (2026)
1. **3-second hold rate** — YouTube explicitly uses this as a quality gate. If viewers leave before 3 seconds, the Short dies.
2. **Swipe-away rate** — Low swipe-away = high quality signal.
3. **Subscribe-after-watch rate** — Shorts that convert viewers to subscribers get massive push.
4. **Watch-to-long-form rate** — If Shorts viewers then watch your long-form, YouTube boosts both.
5. **Comment engagement** — Comment count and sentiment.
6. **Share rate** — External shares (WhatsApp, SMS, etc.) are strongest.

### Best Practices
- 30-58 seconds is the sweet spot (under 30 feels thin, 60 feels long)
- Title is critical — treat it like a search query your audience would type
- First frame must have motion AND text (no black frames, no logos)
- End with CTA to long-form content ("Full breakdown on the channel")
- Use Shorts to test topics — if a Short goes viral, make a 10-minute video on it
- Add #Shorts in description (still helps categorization)
- Consistency matters: Daily Shorts for 30 days = algorithmic trust
- Thumbnail auto-selected — make sure every frame is visually strong

### Optimal Posting Times
- Weekdays: 2-4pm, 7-9pm
- Best days: Friday, Saturday (YouTube has more weekend traffic than other platforms)
- Upload and schedule — consistency > perfect timing

## Cross-Platform Rules

### Content Adaptation (NOT Copy-Paste)
| Platform | Tone | Length | CTA Style |
|----------|------|--------|-----------|
| Instagram | Visual, emotional, aspirational | 30-60s video, 150-word caption | "Save this," "Share with someone" |
| TikTok | Raw, fast, entertaining | 15-60s, keyword-rich caption | "Follow for part 2," "Comment your answer" |
| LinkedIn | Professional, insightful, structured | 1000-1300 char text, 3-min video | "What would you add?", "Agree or disagree?" |
| X | Punchy, bold, quotable | 280 char or 5-7 tweet thread | "Quote tweet your take," "Bookmark this" |
| YouTube Shorts | Educational, hook-heavy, value-dense | 30-58s | "Full video on the channel," "Subscribe for more" |

### Stagger Schedule
Never post the same content on multiple platforms the same day. Stagger:
- Day 1: Instagram Reel
- Day 2: TikTok (slightly different edit/hook)
- Day 3: LinkedIn text version
- Day 4: X thread version
- Day 5: YouTube Short (re-edited for YouTube audience)

### Content Lifespan
| Platform | Peak Reach Window | Evergreen Potential |
|----------|-------------------|---------------------|
| Instagram Reels | 24-48 hours | Low-Medium (search improving) |
| TikTok | 2-7 days (can resurface months later) | High (FYP resurfacing) |
| LinkedIn | 24-72 hours | Low |
| X | 15 minutes - 4 hours | Very Low (unless viral) |
| YouTube Shorts | 7-30 days | High (search + suggestions) |
