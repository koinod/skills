# knowledge-extractor (Lite)

**Capture the knowledge that walks out the door.**

Extract decisions, processes, expertise maps, and tribal knowledge from your meetings, Slack threads, and documents — automatically.

## Quick Start

```bash
openclaw install knowledge-extractor
openclaw run knowledge-extractor extract meeting-transcript.txt
openclaw run knowledge-extractor map
```

## What It Extracts

1. **Decisions** — What was decided, by whom, why
2. **Processes** — Undocumented workflows and tribal knowledge
3. **Expertise** — Who knows what, single points of failure
4. **Insights** — War stories, anti-patterns, heuristics
5. **Action Items** — Commitments, owners, deadlines
