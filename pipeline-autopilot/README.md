# pipeline-autopilot (Lite)

**Your pipeline runs itself.**

Stop checking dashboards. pipeline-autopilot scores your deals, prioritizes your day, and creates recovery plans for stale opportunities.

## Quick Start

```bash
openclaw install pipeline-autopilot
openclaw run pipeline-autopilot ingest deals.csv
openclaw run pipeline-autopilot brief
```

## What It Does

1. **Scores every deal** (0-100) using 6 weighted signals
2. **Generates daily priorities** — who to call, ranked by revenue impact
3. **Recovers stale deals** — breakup emails, new angles, escalation plays
4. **Forecasts revenue** — weighted by deal health, not wishful thinking
