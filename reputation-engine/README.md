# Reputation Engine Lite

**Stop ignoring reviews. Stop responding emotionally. Start responding professionally.**

Reputation Engine Lite gives you framework-driven review responses for any service business. HEARD for negatives, AMPLIFY for positives, BRIDGE for neutrals.

## What's Included (Lite)

- `/respond` -- Generate a professional review response (positive, negative, or neutral)
- `/templates` -- Template library by industry (3 positive, 3 negative, 2 neutral)
- 3 response frameworks (HEARD, AMPLIFY, BRIDGE)
- Industry tone guidelines (5 industries)

## What's in the Full Version

- 8 commands (audit, respond, templates, request-sequence, sentiment, compete, report, crisis)
- Multi-touch review request campaigns (SMS + email sequences by industry)
- Sentiment analysis with theme extraction and urgency flagging
- Competitive reputation benchmarking (side-by-side with competitors)
- Crisis response playbook with severity assessment
- Full legal compliance guidance (FTC, TCPA, HIPAA, Yelp TOS)
- Platform-specific guidelines (Google, Yelp, Facebook, industry-specific)
- Reputation Score formula with weighted factors
