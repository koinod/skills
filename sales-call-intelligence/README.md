# Sales Call Intelligence (Free)

Drop any sales call transcript and get an instant breakdown: talk ratio, SPIN Selling score, objection detection, and a straight A-F grade. No fluff.

## What You Get

- **Talk-to-listen ratio** with monologue flags and phase breakdown
- **SPIN Selling scorecard** (0-100) with line-by-line examples from your transcript
- **Objection detection** — type, root cause, and what a top closer would have said
- **Key topic mapping** — pain points, competitors, stakeholders, next steps
- **Honest grading** — a C is a C, not a "great start!"

## Commands

| Command | What It Does |
|---------|-------------|
| `/analyze` | Full 4-dimension report with SPIN score and action items |
| `/quick` | 60-second grade + one win + one fix |
| `/coach` | Sales manager 1:1 review with practice recommendations |

## Usage

Paste a transcript (any format — plain text, VTT, SRT, JSON) after the command.
